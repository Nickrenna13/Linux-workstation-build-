# Linux Commands I Use Regularly

These are commands I look up often and use while troubleshooting:

- lsblk
- dmesg
- journalctl -xe
- lspci
- lsusb
- top / htop
- df -h
- free -m
- sudo systemctl restart <service>
- sudo apt update && sudo apt upgrade
- sudo dnf update (fedora)


I still look up commands as needed — this list grows as I learn.
