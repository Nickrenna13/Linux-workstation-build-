# Linux Distros Tested

## Pop!_OS
- Best GPU support
- Smooth installation
- Good for development

## Ubuntu Server GUI MATE
- Lightweight
- Good for troubleshooting
- Some manual driver work needed

## Fedora
- Fast and modern
- Required manual configuration for GPU
- Best working OS for my Motherboard with Asus

Testing multiple distros helped me understand how Linux interacts with hardware and the different with package managers (Apt and Dnf).
