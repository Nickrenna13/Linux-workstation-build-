# Linux Workstation Build & Troubleshooting

This repository documents the process of building my personal Linux workstation/server from the ground up. 
The goal is to show real-world problem solving across hardware, BIOS configuration, Linux installation, 
driver issues, and system stability.

I built this system to learn more about Linux, hardware behavior, and how software interacts with physical components. 
This repo includes notes, fixes, and lessons learned along the way.

## What This Repo Covers
- Full hardware parts list
- BIOS configuration and issues encountered
- GPU and driver troubleshooting
- Linux distro testing and installation notes
- Commands and tools I use regularly
- General lessons learned about hardware/software interaction

## Why I Built This
I enjoy hands-on troubleshooting and wanted a deeper understanding of Linux systems. 
This project helped me learn how to diagnose hardware problems, fix boot issues, 
and understand how Linux behaves on different hardware setups.

## Status
Actively updated as I continue learning Linux and experimenting with new tools.
